# NEXUS favicon package

`public/` 直下にこの中身を入れる想定です。

## index.html に入れるタグ

```html
<link rel="icon" href="/favicon.ico" sizes="any" />
<link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png" />
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png" />
<link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png" />
<link rel="manifest" href="/site.webmanifest" />
<meta name="msapplication-config" content="/browserconfig.xml" />
<meta name="msapplication-TileColor" content="#F8FBFC" />
<meta name="theme-color" content="#0B1F2A" />
```

## Included sizes

16, 24, 32, 48, 57, 60, 64, 70, 72, 76, 96, 114, 120, 128, 144, 150, 152, 167, 180, 192, 256, 310, 384, 512, 1024 px.
